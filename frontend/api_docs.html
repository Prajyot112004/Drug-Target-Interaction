DrugTarget AI

API Documentation

Available Endpoints

1. Predict Binding Affinity

POST /predict

Description: Predicts drug-target binding affinity (pKd)

Request Body (JSON): 
{
  "smiles": "CC(=O)Oc1ccccc1C(=O)O",
  "fasta": ">Protein\nSEQUENCEHERE..."
}
Response: 
{
  "affinity_score": 7.85,
  "unit": "pKd",
  "interpretation": "Moderate to Strong Binding Predicted",
  "confidence": "Medium"
}

2. Feature Importance

GET /feature_importance

Returns which molecular and protein features are most important for prediction.

3. Model Evaluation Data

GET /evaluation_data

Returns actual vs predicted values and performance metrics for charts.

4. Frontend

GET / → Main Predictor Page

GET /evaluation.html → Model Evaluation Dashboard
